## Up and Running 

```
$ git clone git@github.com:streamerd/reactOnBoot.git
$ cd reactOnBoot
$ mvn clean install
```
```

or 
$ java -jar target/reactOnBoot-0.0.1-SNAPSHOT.jar
```
```
Swagger Url:
http://localhost:8080/swagger-ui.html
```

### Prerequisites



```
Jdk >= 1.8.x,
Maven >= 3.x
